# Attendance System

## How to Set Up and Run

### 1. Clone the repository:
https://github.com/JesusThushi/attendance-system-repo.git


### 2. Set up the Server:
- Navigate to the `server` folder and run:
- mvn spring-boot:run


### 3. Set up the Client:
- Open `client/login.html` in your browser.
